# Work Log Template: minimal

A compact log — just date, bullets, and a one-line summary. No categories, no week headings, no deliverables section. Good for fast daily notes or when content is light.

---

## Day Entry Blocks (repeat for each date, oldest first)

```
paragraph     — [bold] Date: "4月12日"
bulleted_list_item — bullet (all bullets combined, no category grouping)
bulleted_list_item — bullet
...            (max 6 bullets per day, keep only the most important)
paragraph     — [plain, color: gray] One-line summary
divider
```

---

## Rules

- No `heading_1` / `heading_2` / `heading_3` headings
- No category grouping — all bullets in one flat list
- Max 6 bullets per day, drop the minor ones
- Summary: one sentence, plain, no label prefix

---

## Example

```
**4月12日**
- 实现 SessionEnd hook 自动写入草稿区
- 完成 /work-log 整理逻辑
- 发布 v1.0.0

完成了 work-log skill 的开发和发布。

---
```
